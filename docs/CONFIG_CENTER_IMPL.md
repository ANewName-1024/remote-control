# 配置中心优化方案 - 本地化部署

## 架构

```
┌─────────────────────────────────────────────────────────────────┐
│                     Spring Cloud Config Server                   │
│                        (config-service 改造)                     │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────────────┐    ┌─────────────────────────────────────┐│
│  │  REST API      │    │        config-repo (Git-backed)     ││
│  │  /{app}/{env}  │───>│  ├── user-service.yml              ││
│  │  /{app}/{prof} │    │  ├── gateway.yml                   ││
│  │  /encrypt      │    │  ├── ops-service.yml               ││
│  │  /decrypt     │    │  └── shared/*.yml                   ││
│  └─────────────────┘    └─────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────────┘
          ▲                ▲                ▲                ▲
          │                │                │                │
    ┌─────┴─────┐   ┌─────┴─────┐   ┌─────┴─────┐   ┌───┴────┐
    │ Gateway   │   │   User    │   │   Ops     │   │ Eureka │
    └───────────┘   └───────────┘   └───────────┘   └────────┘
```

## 实施步骤

### Phase 1: 改造 config-service 为 Config Server

1. 添加依赖
2. 启用配置服务器
3. 配置 config-repo 为后端

### Phase 2: 各服务配置

1. 添加 config client 依赖
2. 配置 bootstrap.yml
3. 保留本地必要配置

### Phase 3: 动态刷新

1. 启用 actuator refresh
2. 配置 @RefreshScope
